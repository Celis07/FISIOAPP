# FisioApp 🏃‍♂️

Panel de gestión para fisioterapeutas.

## Cómo publicar en Vercel (sin saber programar)

### Paso 1 — Sube a GitHub
1. Ve a github.com y crea una cuenta
2. Clic en "New repository" → nombre: `fisioapp` → "Create repository"
3. En la página del repo, clic en "uploading an existing file"
4. Arrastra TODA la carpeta `fisioapp` (o los archivos uno por uno)
5. Clic en "Commit changes"

### Paso 2 — Despliega en Vercel
1. Ve a vercel.com → "Sign up with GitHub"
2. Clic en "Add New Project"
3. Selecciona el repositorio `fisioapp`
4. Clic en "Deploy" (Vercel detecta todo automáticamente)
5. ¡Listo! En 2 minutos tienes tu link 🎉
